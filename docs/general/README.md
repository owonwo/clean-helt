## General Concerns

### Base URL

**`Base URL: /api`**

### Accessing Entities

**`GET /entity/{entity?}`**

Returns a list of specified entity - if available and open. If entity is not specified,
it returns a list of all open entities

Sample Data:

```
{entity} => [doctors|hospitals|pharmacies|laboratories|optional] 
```
